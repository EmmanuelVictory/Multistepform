# 📝 Multi-Step Form

## 🧠 About This Project

I built this Multi-Step Form to improve my understanding of user flow and state management across multiple UI sections.

Long forms can feel overwhelming, so I wanted to design something that improves user experience by breaking the process into smaller, manageable steps.



## 🎯 Project Goals

- Practice form validation
- Manage multi-step state transitions
- Improve UX thinking
- Implement progress tracking



## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript



## ⚙️ How It Works

1. Form is divided into multiple steps.
2. JavaScript tracks the current step index.
3. Validation runs before moving to next step.
4. Final step shows summary before submission.

All transitions happen dynamically.



## 💡 What I Learned

- Managing state across UI sections
- Validating inputs properly
- Preventing data loss between steps
- Improving UX structure



## 😅 Challenges I Faced

- Preventing users from skipping required fields
- Maintaining smooth navigation
- Resetting form state properly



## 🚀 Future Improvements

- Save progress to local storage
- Add animation transitions
- Connect to backend database



## 🏆 Why This Project Matters to Me

This project improved how I think about user experience and form structure.

It helped me understand that functionality and UX go hand in hand.
